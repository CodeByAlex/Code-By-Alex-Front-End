A talk about developing with the EmailJS Service to help send emails from a REST endpoint. As a part of this presentation, I went over how to make rest calls to the EmailJS API, set up templates for uniform email messaging, and an example of how I use the service for my personal website.

Developing sites for User Inclusivity: A Talk About Web Accessibility

Audience: CharmCityJS Meetup Group

A talk about accessibility that covered the reasons why developers should strive to meet Accessibility Standards, such as WCAG. Testing tools and best development practices were described with several real-world examples. I then went over the business value for adding the necessary features so that the members of the audience could return to their jobs with ways to backup the process change.

Hello Cloud Workshop

Audience: Towson University

A workshop that was an extension of my Introduction to Angular talk. During the workshop, Saurabh Khatiwada and I went over the fundamentals of the Angular framework, instructed students on how to upload their code to Github, then deploy that code to an AWS S3 bucket. The students could then view their live website in the browser by hosting their code in the cloud.

Scheduling and Path Planning for Computational Ferrying

Audience: G2 Inc.

A presentation and overview of the IEEE published paper written by Sebasti√°n Zanlongo, Leonardo Bobadilla, Tamim Sookoor, and me. In this paper, we extend previous work on computational ferrying, where Mobile High Performance Computers (MHPCs) physically move the necessary hardware into the proximity of mobile devices. Our extension proposed several improvements: 1) Path planning which yields more accurate scheduling; 2) Collision checking, which permits robots to carry out the role of MHPCs; 3) Prioritization, allowing operators to assign weights to tasks.
